# Sqeets Shop

Sweets Shop App is sweets gallery app using LINE LIFF and message API.

## Architecture

- Frontend: React, LINE Front-end Framework (LIFF) as static web app
- Backend: Laravel (PHP), MySQL

## Features

- User Registration / Login via LINE Login
- User can earn / spend points when they use the shop's services
- Store manager add sweets and update stock

## MUST Conditions

- User must login and must be a friends of the shop's official account to use the app.
