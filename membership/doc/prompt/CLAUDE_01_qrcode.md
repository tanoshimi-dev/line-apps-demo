# Point earning, spending, function

## Earning Points

User Operation

1. Click ポイントを貯める (Earn Points)
2. QRコードをスキャン (Scan QR Code)
3. ポイント獲得完了 (Points Earned)

## Spending Points

User Operation

1. QRコードをスキャン (Scan QR Code)
2. 利用ポイントの入力
3. ポイント利用確認 (Confirm Point Usage)
4. ポイント獲得完了 (Points Earned)
5. 利用ポイントの表示 (Display Used Points)
