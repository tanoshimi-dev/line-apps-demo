# Salon Reservation

Salon Reservation App allows users to reserve salon services using LINE LIFF and message API.

## Architecture

- Frontend: React, LINE Front-end Framework (LIFF) as static web app
- Backend: Laravel (PHP), MySQL

### Operations

User Operation

1. Register / Login
2. View Available Services, calendar, and staff
3. Make Reservation

- Select service, date, time, and staff
- Confirm reservation details
- Receive reservation confirmation
